_G.create_follow_camera (id, target_object_id, follow_offset, aim_offset)
追尾カメラを生成

Parameters:

  • id string カメラID(switch_cameraで使うやつ)
  • target_object_id string 追尾対象オブジェクトID
  • follow_offset {x:number, y:number, z:number} 追尾対象に対する位置 デフォルト:{x = 0, y = 4, z = -8}
  • aim_offset {x:number, y:number, z:number} 映す相対位置 デフォルト:{x = 0, y = 0, z = 0}（キャラクターだと足元）
